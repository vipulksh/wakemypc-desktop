# wakemypc-desktop

Desktop GUI companion for [wakemypc.com](https://wakemypc.com). Same job as `wakemypc` (the CLI) but with a window instead of a terminal — handy when you've got a Pico that needs a quick WiFi tweak and you don't want to remember flag names.

## Install

Direct from GitHub:

```bash
pip install git+https://github.com/wakemypc/wakemypc-desktop.git
wakemypc-desktop
```

Linux: same `dialout` group caveat as the CLI.
